# Cardinal Calculator

A comprehensive financial calculator with Stanford Cardinal branding for Time Value of Money (TVM) calculations.

## Features

✅ **Complete TVM Calculations** - Solve for any variable:
- Present Value (PV)
- Future Value (FV)
- Payment (PMT)
- Number of Periods (n)
- Interest Rate

✅ **Compounding Frequencies** - Annual, Semi-Annual, Quarterly, Monthly, Weekly, Daily

✅ **Step-by-Step Solutions** - See the mathematical formulas and calculations

✅ **Save Results** - Keep track of your previous calculations with localStorage

✅ **Scientific Calculator** - Quick calculator with memory functions built-in

## Live Demo

Visit: **https://matthewdlang18.github.io/cardinal-calculator/**

## How to Use

1. Select which variable you want to solve for
2. Enter the known values
3. Click "Calculate" to see the result
4. View the step-by-step solution with formulas
5. Save results for future reference

## Local Development

Simply open `index.html` in any modern web browser. No build process or dependencies required!

## Technologies

- Pure HTML/CSS/JavaScript
- No external dependencies
- Works offline after initial load
- localStorage for data persistence
